1. Buat variabel "m" = 600
2. Buat variabel "a" = 2
3. Buat variabel "resultan" = ''
4. Hitung nilai "m" x "a"
5. Masukkan hasil hitungan no.4 ke dalam variabel "resultan"
6. Tampilkan hasil nilai "resultan"

Pseudocode :

STORE "m" with 600
STORE "a" with 2
STORE "resultan" without any value

CALCULATE "m" times "a"
SET "resultan" value with calculation result

DISPLAY "resultan"

Algoritma :

1. Buat variabel "tahun1" = ''
2. Hitung nilai "tahun1" dengan ketentuan habis di bagi 4
3. Buat variabel "remainder0" dengan nilai hasil dari no.2
4. Hitung nilai "tahun1" dengan ketentuan tidak habis dibagi 100
5. Buat variabel "remainder1" dengan nilai hasil dari no.4
6. Buat variabel tahun kabisat1 dengan ketentuan jika "remainder0" sama dengan 0 dan Jika "remainder1" tidak sama dengan 0 maka tampilkan "tahun kabisat" jika tidak tampilkan "bukan tahun kabisat".
7. Buat variabel "tahun2" = ''
8. Hitung nilai "tahun2" dengan ketentuan habis di bagi 4
9. Buat variabel "remainder2" dengan nilai hasil dari no.8
10. Hitung nilai "tahun2" dengan ketentuan habis dibagi 100
11. Buat variabel "remainder3" dengan nilai hasil dari no.10
12. Hitung nilai "tahun2" dengan ketentuan habis dibagi 400
11. Buat variabel "remainder4" dengan nilai hasil dari no.12
12. Buat variabel tahun kabisat2 dengan ketentuan jika "remainder2", "remainder3" dan "remainder4" sama dengan 0 maka tampilkan "tahun kabisat" jika tidak tampilkan "bukan tahun kabisat".


Pseudocode :

STORE "tahun1" to any value
DISPLAY "tahun1"
CALCULATE "tahun1" mod 4
STORE "remainder0" to the result of calculation
THEN
CALCULATE "tahun1" mod 100
STORE "remainder1" to the result of calculation
IF "remainder0" equals to 0
AND
IF "remainder1" not equals to 0
DISPLAY "TAHUN KABISAT"
ELSE
DISPLAY "BUKAN TAHUN KABISAT"

STORE "tahun2" to any value
DISPLAY "tahun2"
CALCULATE "tahun2" mod 4
STORE "remainder2" to the result of calculation
THEN
CALCULATE "tahun2" mod 100
STORE "remainder3" to the result of calculation
THEN
CALCULATE "tahun2" mod 400
STORE "remainder4" to the result of calculation
IF "remainder2" equals to 0
AND
IF "remainder3" equals to 0
AND
IF "remainder4" equals to 0
DISPLAY "TAHUN KABISAT"
ELSE
DISPLAY "BUKAN TAHUN KABISAT"

Algoritma :

1. Buat variabel "baju" = ''
2. Hitung variabel "baju" setiap baju yang dimasukkan ke dalam mesin cuci maka nila "baju" ditambah 1
3. Jika nilai baju sudah mencapai angka 20 maka tampilkan "jalankan mesin cuci"

Pseudeucode:

STORE "baju" with 0
WHILE "baju" <= 20
ADD "baju" by 1
DISPLAY "jalankan mesin cuci"

Algoritma :

1. Buat variabel "murid" = 0
2. Buat variabel "kuku" = ''
3. Hitung variabel "murid" jika nilainya kurang dari 40 atau belum sama dengan 40 maka ditambah 1
4. Masukkan nilai variabel "kuku" jika nilai nya sama dengan 0 maka tampilkan "dipuji", jika tidak maka tampilkan "dihukum"

Pseudocode:

STORE "murid" with 0
STORE "kuku" to any number
WHILE "murid" <= 40
ADD "murid" by 1
THEN
IF "kuku" = 0
DO "dipuji"
ELSE
DO "dihukum"
